Тестовое задание на front-end:
	Проект – постройка виртуальной экосистемы, небольшого мира, населенного существами, которые двигаются и борются за выживание.
	Мир будет двумерной сеткой, где каждая сущность занимает одну клетку. На каждом ходу существа получат возможность выполнить какое-либо действие.

Символы:
	# - стена, не движется, выполняет роль препятствия
	' '(пробел) - пустое пространство, на которое можно стать
	o - травоядное существо, умеет ходить в любом направлении, "есть" траву, размножаться
	* - трава - служит едой для травоядных, умеет размножаться
	@ - хищное существо, поедает травоядных, не ест траву, умеет размножаться

Условия:
	С заданной задержкой выполняется ход, каждое живое существо двигается в случайном направлении, ест свою еду, если она в поле зрения. Поле зрения любой сущности - 1 клетка вокруг существа. У каждой сущности есть запас энергии, который пополняется во время еды. Количество энергии существа, которое ест пополняется на количество энергии существа, которое оно ест, а количество энергии второго существа становится 0, т.е. существо умирает. 
	Каждый ход количество энергии существа, которое сделало шаг, уменьшается на 1. Если количество энергии равно 0, то существо умирает. Если количество энергии достигает заданного значения, то в случайной соседней клетке появляется существо этого же типа, а энергия "родителя" делится на 2 и "дочернее" существо получает эту половину энергии от родителя.

	1) grid.js - Изображение пространства. 
		- cоздание объекта характеризуещего сетку
		- cоздание координат движения

	2) world.js - Мировой объект.
		- вспомогательные функций 
		- объект World c методами
		- объект Wall
		- turn (ход)
		- возможность движения, метод letAct и checkDestination
		- объект для вычисления координат View + методы

	3) animateworld.js - Анимаирование.
		- файл скачан с интернета

	4) lifelikeworld.js - наследуем с помощью prototype объект World.
		- добавление возможностей еды и размножения
		- методы grow, move, eat, reproduceы
